# Unity Tools

### [Vertex Data Baker](https://github.com/kelo221/AnimationTextureBaker)  
Bake animations into a single mesh that is controller by a script - very performant!

### [Cheap URP lighting](https://github.com/kelo221/DynamicLighting-URP)  
Old-school styled baked lighting with unique shading system for dynamic objects.

### [BIRP Motion Blur](https://github.com/kelo221/KinoMotion-Modernized)  
Old-school styled baked lighting with unique shading system for dynamic objects.

### [Recycle GameObjects with ease](https://github.com/kelo221/RecyclerKit-Modernized)
Provides an easy to use interface to recycling and prewarming game object.

### [Decals for BIRP - focus on Henry's Lighting Package](https://github.com/kelo221/DynamicDecals-DynaLight)
Projected decals for BIRP.
